# Olvasd be a fájl tartalmát egy olyan listába, amelyben minden sor egy szótárban van tárolva. Tehát a lista elemei szótárok.

Az első sort a fejlécben tárold (header). A fejléc csak egy lista, amelyben az oszlopok nevei vannak sztringként. Vagyis az első sor sztringek listája lesz.
A második sortól vannak az adatok. Ezek lesznek a föntebb leírt szótárokat tartalmazó listában.

# A program írja ki:
### Az iskolába járó fiú tanulók összlétszámát!
### Hány osztályfőnöknek tartalmaz a neve "Nagy" szót.
### Hány 5.-es diák tanul az iskolában (5. évfolyam tanulói)?
### Hogy hívják annak az osztálynak az osztályfőnökét, amelyiknek a legkevesebb tanulója van?
### Hány osztálynak van 4-esnél rosszabb tanulmányi átlaga?
### A "Judit" nevű osztályfőnökök osztályában hányan tanulnak összesen?
### Mely osztályokban azonos a fiúk és a lányok száma? A megfelelő osztályok neveit írja ki a program: pl.: 8.b, 9.a
### Hány osztályban tanul több lány, mint fiú?
